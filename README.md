# Jupyter Exploration

This repository contains materials for an introductory lab on GitHub and Jupyter Notebooks. The lab covers repository creation, basic version control concepts, Markdown formatting, and execution of Python code in a Jupyter Notebook environment.

## Contents

- README.md
- My_First_Notebook.ipynb
- Journal/Summary document

## Lab Session Goals

- Create and make commits to New GitHub Repository
- Create a Jupyter Notebook
- Use Markdown and Python within notebook
- Upload notebook files to GitHub
